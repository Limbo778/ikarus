Название проекта: Yancuic Tlanextia 
Визуальный стиль и интерфейс
Цветовая палитра: Используются мягкие современные оттенки фиолетового и черного с плавными градиентами, создающими ощущение глубины и технологичности. Цветовая схема подчеркивает минимализм и стиль, оставаясь приятной для глаз даже при длительном использовании.
Типографика: Применяются современные шрифты без засечек (например, Montserrat или аналогичные), обеспечивающие высокую читабельность и элегантность текста на всех устройствах.
Анимация: Интерфейс включает мягкие и плавные переходы, ненавязчивые анимации (например, при загрузке или переключении экранов), что делает взаимодействие с платформой интуитивно понятным и комфортным.
Философия пользовательского интерфейса: Основной акцент сделан на легкости восприятия, минималистичном дизайне и выделении контента. Современный стиль сочетается с функциональностью, избегая перегруженности элементами.
Язык сервиса: Полностью на русском языке, что обеспечивает доступность для русскоязычной аудитории.
Главная страница
Экран приветствия/входа
Логотип: Центральный элемент — стилизованная маска бога Шипе-Тотека (Xipe Totec), отражающая культурные корни названия и символизирующая обновление и связь с традициями в современном контексте.
Авторизация администратора:
Кнопка "Войти как администратор" открывает форму для ввода логина и пароля.
Данные для входа выдаются через Telegram-бот, обеспечивая безопасный и удобный доступ.
При первом запуске сервиса отображается кнопка "Подключить Telegram-бот" для настройки API (кнопка исчезает после успешной интеграции).
Интерфейс для пользователей:
Поля для ввода имени и ID конференции, позволяющие гостям быстро подключиться к мероприятию.
Личный кабинет администратора
Функционал:
Кнопка "Создать конференцию": Запускает процесс настройки новой видеовстречи.
Список конференций: Отображает текущие и завершенные мероприятия с деталями:
Статистика (количество участников, продолжительность и т.д.).
Кнопка "Подключиться" (сохраняется информация об аккаунте, с которого выполнен вход).
Кнопка "Скачать запись" (доступна только для завершенных конференций).
Кнопка "Завершить конференцию" (принудительное окончание).
Общая статистика сервиса: Данные о нагрузке, количестве проведенных встреч и других метриках.
Имя администратора: Персонализированное приветствие.
Кнопка "Выйти": Завершение сессии администратора.
Детали создания конференции
Процесс:
Ввод названия конференции в специальном окне.
После создания генерируются:
Ссылка-приглашение для распространения среди участников.
Уникальный код комнаты для ввода на главной странице.
Кнопка "Подключиться" для немедленного входа.
Особенность: Администратор, авторизованный на главной странице, сохраняет свои привилегии при входе в конференцию любым способом (по ссылке или коду).
Конференц-зал
Интерфейс участников:
Вид в стиле сетки (аналог Jitsi Meet): маленькие окна участников справа, основной фокус — на администраторе.
Аватары участников: первая буква имени на аватаре, полное имя — внизу бокса.
Кнопки и функции:
Для гостей: Стандартный набор: отключение микрофона/камеры, чат, демонстрация экрана, поднятие руки и т.д.
Для администратора: Дополнительные опции:
Кнопка "Завершить конференцию".
Автоматическая запись (активируется при старте).
Значок записи в углу экрана для информирования участников.
Авторизация и доступ
Гости:
Два способа входа:
Переход по ссылке-приглашению.
Ввод кода комнаты на главной странице.
После выбора способа предлагается ввести имя, затем пользователь мгновенно попадает в конференцию.
Администраторы:
Авторизация только на главной странице через логин и пароль от Telegram-бота.
Учетная запись действительна для всех функций сервиса.
Вход в конференцию возможен любым способом (по ссылке или коду) без повторной авторизации.
Видеоконференции
Технология: Использование WebRTC для обеспечения многопользовательских встреч с низкой задержкой.
Функционал: Подобен Jitsi Meet:
Отключение микрофона/камеры.
Чат.
Демонстрация экрана.
Поднятие руки.
Серверная обработка: Потоковая передача и перекодирование видео/аудио на стороне сервера для стабильности и качества.
Улучшение качества звука и видео
Технологии ИИ:
Подавление шума.
Повышение четкости изображения.
Цветокоррекция.
Масштабирование видео в реальном времени.
Особенность: Обработка выполняется исключительно на сервере, что минимизирует нагрузку на устройства пользователей и обеспечивает одинаково высокое качество при любых условиях подключения.
Запись и сохранение
Механизм:
Автоматическая запись активируется администратором при запуске конференции.
После завершения запись сохраняется и становится доступной для скачивания в личном кабинете администратора.
Telegram-бот
Функции:
Выдача логинов и паролей администраторам через защищенный канал.
Уведомления о создании и завершении конференций для оперативного информирования.
Основные моменты
Загрузка сервера: Все процессы (обработка видео, звука, ИИ-функции) выполняются на сервере, что снижает требования к клиентским устройствам.
Совместимость: Интерфейс полностью адаптирован для работы на любых устройствах (ПК, планшеты, смартфоны).
ИИ-улучшения: Используются легкие и оптимизированные модели для обработки мультимедиа в реальном времени, что обеспечивает высокую скорость и качество.
